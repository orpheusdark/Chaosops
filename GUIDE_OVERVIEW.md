## 🔧 System Overview

```
CHAOSOPS RL TRAINING PIPELINE
═════════════════════════════════════════════════════════════

INPUT (Google Colab)
    ↓
CLONE: GitHub repo
    ↓
INSTALL: Qwen2.5-0.5B, LoRA, training libs
    ↓
INITIALIZE: ChaosOpsEnv (RL environment)
    ↓
TRAIN (10 grouped episodes)
    • Load Qwen2.5-0.5B (4-bit quantized)
    • Setup LoRA adapters (0.1% parameters)
    • Run episodes, collect trajectories
    • Compute GRPO-style advantages
    • Update LoRA weights
    • Save adapter → chaosops-qwen-grpo/
    ↓
EVALUATE (60 test episodes)
    • Baseline: Random policy (20 episodes)
    • Trained: With LoRA adapter (20 episodes)
    • Variation: Distribution shift test (20 episodes)
    ↓
ANALYZE
    • Compare metrics (success, reward, steps)
    • Compute improvements
    • Generate verdict (LEARNING? SCRIPTED? ROBUST?)
    ↓
VISUALIZE
    • Dashboard charts (4 comparison plots)
    • Save to training_results_dashboard.png
    ↓
OUTPUT (Google Colab)
    • chaosops-qwen-grpo/ (trained adapter)
    • training_results_dashboard.png (visualization)
    • eval_results (printed JSON)
    ↓
DEPLOY
    • Auto-sync to GitHub
    • Auto-sync to HF Space
    • Backup to Google Drive (optional)
```
